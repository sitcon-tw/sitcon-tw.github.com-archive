學生計算機年會

理念

學生的身分，對你而言代表著什麼？

在日常生活中，你可曾有對資訊科學的憧憬，希望能認識更多同好、吸收新知？或是已經憑藉自身技藝遨遊於天際，在尋找展現自己的舞台、分享實務經驗？學生計算機年會能夠滿足你的一切！以學生為本、由學生自發舉辦，學生計算機年會（SITCON）不只是學生「學以致用、教學相長」的實際展現，更冀望所有對資訊有興趣的學生，能夠在年會裏齊聚一堂，彼此激盪、傳承、啟發。

時間 & 地點

2014 年 3 月 15 日

中央研究院 人文社會科學館

Call for papers 學生與創新實作

主題範疇

遊戲

  • 遊戲程式設計
  • 美術/音樂創作
  • 遊戲模組

範例 RPG Maker & Ruby、Ren’py 與視覺小說創作、線上遊戲 AI 解析

程式語言

  • 軟體設計模式
  • 程式語言概念
    （如物件導向、函數式程式設計、非同步模式）
  • 合作經驗
  • 程式語言學習

    • 範例 Javascript Prototype、Functional Programming in Ruby、我與 Verilog 的奮鬥、Debugging Asynchronous Program、軟體測試入門

Web

  • 各種網頁技術（前端、後端）
  • 網頁設計模式
  • 美術設計
  • 工作經驗

範例 Angular.js 入門、Django 效能調校、五個步驟學會與設計師溝通、網頁開發市場現況

其他

舉凡 Pair Programming 心得、 學生網管的一天；或是畢業專題甘苦談、我的程式競賽血淚史、第一次接案就上手；甚至於校園推廣 Open Source、學校門禁系統安全實測、建構 Email 伺服器的心路歷程……

只要與資訊技術、電腦相關的講題，仍然歡迎投稿！

議程種類

Talk / Short Talk

Talk 為 40 分鐘的議程，加上10分鐘的問答時間；

Short Talk 為 10 分鐘的議程，問答時間由講者自行安排。兩者僅有時間上的差別。

Lightning Talk

除了正規議程外，SITCON 2014 亦開放五個名額的 Lightning Talk，顧名思義即為閃電一樣短的演講，每位講者僅有3分鐘，所以請拿捏好時間，3分鐘一到可是會被趕下臺的！

大會當天早上開放報名，先搶先贏！

投稿方式

稿件均經議程組形式審查，交予該領域之審稿委員匿名審稿；審稿委員由業界人士、學界人士及 SITCON 議程組員組成，選定契合本屆年會主旨者優先錄取，稿件一律以速審為原則。

期限

  • 2014 年 1 月 24 日 午夜投稿截止
  • 2014 年 2 月上旬 審稿結束
  • 2014 年 2 月中旬 公告完整議程

格式

投稿格式如下（Talk 與 Short Talk 的投稿格式相同）

題目
必填，15 字以下
投稿類型
請寫明所欲投稿者為 Talk 或 Short Talk
暱稱
公開於大會手冊及網站，如未填寫，將使用本名
個人介紹
必繳，含標點符號 50 ~ 100 字，安排於大會手冊及網站
摘要
必繳，含標點符號 150 ~ 250 字，安排於大會手冊及網站
詳細說明
不限字數，可斟酌繳交，或附上程式碼、投影片草稿等，以利審稿委員更瞭解你的作品。
聯絡資訊
必填，含真實姓名、手機號碼、電子信箱、所屬學校或單位、職稱或系級，僅供大會聯絡及印製感謝狀使用

請將上述資訊 email 至 SITCON 議程組信箱（session@sitcon.org），有任何疑問也可以 email 至同一信箱詢問。

備註

  1. 為鼓勵投稿，大會將提供投稿人一組 VIP Code，可優先報名參與本年會。
  2. SITCON 議程組得經作者同意，轉換 Talk 或 Short Talk 稿件之投稿類型。
  3. 無論有無入選，仍然歡迎於年會當天報名 Lightning Talk！
  4. 凡稿件經接受者，年會將頒予感謝狀以表彰投稿人之熱情付出與貢獻。
